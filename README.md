# Health Dashboard

Your personal longevity dashboard. Track blood work, hormones, vitamins, and more — all in one place.

---

## Deploy Your Dashboard

**Step 1: Create a Supabase project**
1. Go to [supabase.com](https://supabase.com) → New project
2. Once created, go to **Project Settings → API** and copy:
   - **Project URL** (looks like `https://xxxxx.supabase.co`)
   - **service_role** key (under "Project API keys")

**Step 2: Deploy to Vercel**

Click the button below. You'll be asked to paste the two values from Step 1, then Vercel deploys everything automatically.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/thebodyhqtest/health-dashboard&env=SUPABASE_URL,SUPABASE_SERVICE_KEY&envDescription=From%20Supabase%20%E2%86%92%20Project%20Settings%20%E2%86%92%20API&envLink=https://supabase.com/dashboard)

**Step 3: Finish setup at `/admin`**

Once deployed, open your Vercel URL and go to `/admin.html`. The setup wizard walks you through the rest — no coding required.

---

That's it. Your dashboard is live.
